BYON Ranking Clean Static Version

Isi folder:
1. index.html
2. style.css
3. main.js

Cara pakai:
1. Buka folder ini di VS Code.
2. Jalankan index.html dengan Live Server.
3. Edit tampilan di style.css.
4. Edit data ranking di bagian const rankings dalam main.js.

Catatan:
Versi ini sudah dibersihkan dari struktur WordPress, Elementor, WP Rocket, Google Tag Manager, Meta Pixel, Clarity, dan script tracking lain. Jadi file lebih mudah diedit untuk redesign awal. Setelah desain final, baru pindahkan ke WordPress sebagai template atau embed ke Elementor.
